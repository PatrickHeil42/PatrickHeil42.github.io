Site Map

This site is organized into 3 pages.

Home

The page you are on currently is the home page, or landing page. It displays information about the site and it's features. It can be accessed by selecting Home on the navigation bar at the top of the screen.

Weather

The second page is the Weather Page. It displays a weather forecast for your location using the TomTom API and the OpenWeatherMap API. This page can be accessed by selecting Weather on the navigation bar.
