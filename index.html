Home

Welcome to my personal website! My name is Patrick; I am studying computer science at Miami University.
I made this site as a portfolio and for some web development practice. Some of these projects began from class assignments; others I made on my own.
This site and all it's code is posted on my github.


Site Map


Home

The page you are on currently is the home page. It displays information about the site and it's features. It can be accessed by selecting Home on the navigation bar at the top of the screen.

Weather

The second page is the Weather Page. It displays a weather forecast for your location using the TomTom API and the OpenWeatherMap API. This page can be accessed by selecting Weather on the navigation bar.

Satellite

This project took the longest by far but I'm happy with how it turned out. I used NASA's Worldview site to fetch satelite images of the searched city. If I had a server with the ability to run AJAX I'd load the images directly on the page but since we're on Github Pages users must click the link itself. Images are refreshed daily (except for the clear view, which refreshes every ten minutes!). All images courtesy of NASA.

P5.js

The fourth page is a Javascript project I call Celtic Circles. Using the p5.js library, I made a 5x5 grid where the cells of first row and first column contain spinning circles. Each other cell draws a shape using the x and y from it's row/column circles. The shapes it produced reminded me of Celtic Knots, hence the name).
