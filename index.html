Site Map

This site is organized into 4 pages.

Home

The page you are on currently is the home page, or landing page. It displays information about the site and it's features. It can be accessed by selecting Home on the navigation bar at the top of the screen.

Weather

The second page is the Weather Page. It displays a weather forecast for your location using the TomTom API and the OpenWeatherMap API. This page can be accessed by selecting Weather on the navigation bar.

Satellite

This project took the longest by far but I'm happy with how it turned out. Ever since I started working with API's I've wanted to find a public satellite imagery API and hook something up to it. I got caught up trying to find a free API I could use, missing the easy solution: the snapshot tool from NASA's Worldview site. If I had a server with the ability to run AJAX I'd load the images directly on the page but since we're on Github Pages you'll need to click the links yourself. Images are refreshed daily (except for the clear view, which refreshes every ten minutes!). All images courtesy of NASA.

P5.js

The fourth page is a javascript (art?) project. I made it using the p5.js library; it draws circles using a grid.
